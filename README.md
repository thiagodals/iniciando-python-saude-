# 🧠 Iniciando em Python: Saúde

Projetos simples em Python voltados para saúde e bem-estar.  
Desenvolvidos como prática de programação para iniciantes.

## 📌 Projetos

### 💧 Consumo Diário de Água
Calcula a quantidade ideal de água por dia com base no peso.

Arquivo: `agua_diaria.py`

---

### ⚖️ Calculadora de IMC
Calcula o IMC e classifica o resultado.

Arquivo: `calculadora_imc.py`

---

### 🔥 Calculadora de TMB
Calcula a Taxa de Metabolismo Basal com base em peso, idade e sexo.

Arquivo: `calculadora_tbm.py`

---

## 🚀 Como executar

1. Instale o Python
2. Execute no terminal:

```bash
python nome_do_arquivo.py
